# COMSC-178  Cloud Machine Learning

Notes and course materials from Spring semester 2026.


## Course description

> This intermediate-level course explores the application of the fundamental principles of data science to data collection, statistical inference, predictive modeling, and making decisions with data.
> Techniques for data transformation, regression, classification, clustering, and visualization will
> be explored in the context of developing proficiency in data analysis.
> Students will build and deploy a custom machine learning model on the Amazon AWS cloud platform.
